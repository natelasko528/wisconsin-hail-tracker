# GitHub Repository - READY TO CREATE

## Quick Steps

1. Create .gitignore file (manually in your editor):
```
node_modules/
.env
.env.local
.next/
out/
build/
dist/
*.log
.DS_Store
```

2. Initialize Git:
```bash
cd wisconsin-hail-tracker
git init
git add .
git commit -m "Initial commit"
```

3. Create GitHub repo and push:
```bash
gh repo create wisconsin-hail-tracker --public --source=. --push
```

Or manually at https://github.com/new
